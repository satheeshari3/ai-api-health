This file defines standardized rules to ensure consistent and effective AI prompts.\
\
## General Rules\
1. Always include:\
   - API endpoint\
   - Error code or response time\
   - Context (recent changes or logs if available)\
\
2. Use structured prompts:\
   - Input: JSON log or status code\
   - Output: Human-readable explanation + recommendations\
\
3. Keep explanations concise and actionable:\
   - Max 5 sentences\
   - Include cause + recommended fix\
   - Avoid speculation\
\
4. Maintain tone:\
   - Professional and neutral\
   - Developer-focused language\
\
## Example Prompt Template\
\
\pard\pardeftab720\sa240\partightenfactor0

\f1\fs24 \cf0 Endpoint: [API_URL]\uc0\u8232 Status: [ERROR_CODE]\u8232 Response Time: [TIME_MS]\
Explain the failure and provide actionable suggestions.\
